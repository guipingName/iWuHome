&emsp;&emsp;您也可以在设备列表左滑设备，快速控制设备。比如空调开机。

<img src="../images/device/快速控制.png" width = "300" height = "534">

> 说明：
> 
> 如果空调当前状态是**开机**，左滑时只能看到“关机”按钮。
> 
> 如果空调当前状态是**关机**，左滑时只能看到“开机”按钮。
