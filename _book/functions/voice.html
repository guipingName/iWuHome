语音控制

语音控制

  您可以通过语音来控制智能设备、执行情景模式以及标签。

  您可以按照下面列出的语音格式操作，提高语音识别的效率。

您可以这样说：

控制单个设备

空调

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机 + 设备名称”。

  比如：“打开空调” 或 “关闭空调”。

控制温度

  您可以直接设置空调温度，也可以调高/调低温度，您可以说“调高/调大/大/加/调低/调小/小/减 + 设备名称”。

  比如：“空调30摄氏度” 或 “调高空调”。

控制模式

  您可以说“设备名称 + 制热/自动/送风/除湿/制冷/扫风”。

  比如：“空调制热” 或 “空调除湿”。

插座

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机 + 设备名称”。

  比如：“开启插座” 或 “关闭插座”。

暖气阀

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机 + 设备名称”。

  比如：“打开暖气阀” 或 “关闭暖气阀”。

控制温度

  您可以直接设置暖气阀温度，也可以调高/调低温度，您可以说“调高/调大/大/加/调低/调小/小/减 + 设备名称”。

  比如：“暖气阀30摄氏度” 或 “调高暖气阀”。

灯

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/熄/灭/关机/关 + 设备名称”。

  比如：“打开调光灯” 或 “关闭调光灯”。

控制亮度

  您可以说“调亮/调暗/亮/暗/亮起来/暗下去/暗下来/升高/调高/调大/大/加/降低/调低/调小/小/减 + 设备名称”。

  比如：“调亮灯” 或 “调暗灯”。

窗帘

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关+设备名称”。

  比如：“打开窗帘” 或 “关闭窗帘”。

调整窗帘

  您可以说“升高、调高、调大、大、加、降低、调低、调小、小、减 + 设备名称”。

  比如：”调高窗帘” 或 “降低窗帘“。

电视、机顶盒

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关/不看/看 + 设备名称”。

  比如：“打开电视” 或 “关闭电视”。

换台

  您可以说“设备名称 + 上一个台/下一个台/上个台/下个台/上一个频道/下一个频道/上个频道/下个频道/频道加/频道减/换台/换频道/切换频道”。

  比如：“机顶盒下一个台” 或 “机顶盒频道加”。

音量调节

  您可以说“设备名称 + 取消静音/关闭静音/恢复声音/静音/升高/调高/调大/大/加/降低/调低/调小/小/减”。

  比如：“机顶盒静音” 或 “机顶盒取消静音” 或 “机顶盒调高”。

控制风扇灯

打开/关闭

  如果要打开/关闭风扇灯，您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 设备名称”。

  比如：“开启风扇灯” 或 “关闭风扇灯”。

  如果要控制风速，您可以说“低风速/中风速/高风速 + 设备名称”。

  比如：”风扇灯低风速” 或 “风扇灯高风速“。

遥控器

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 设备名称”。

  比如：”打开遥控器” 或 “关闭遥控器“。

亮度调节

  您可以说“设备名称 + 调亮/调暗/亮/暗/亮起来/暗下去/暗下来/灭/熄/升高/调高/调大/大/加/降低/调低/调小/小/减”。

  比如：”遥控器调亮” 或 “遥控器调暗“。

智能开关

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 设备名称”。

  比如：”打开智能开关” 或 “关闭智能开关“。

声光报警器

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 设备名称”。

  比如：”打开声光报警器” 或 “关闭声光报警器“。

开关面板

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 设备名称”。

  比如：”打开开关面板” 或 “关闭开关面板“。

说明：这样操作后开关面板会全部打开或全部关闭。

  如果您想控制某一路，您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 设备名称 + 某一路的名称”或"打开/开启/开机/开/关闭/停止/关掉/关机/关 + 某一路的名称"

  比如：”打开开关面板第1路” 或 “打开第一路“。

智能门磁

布防/撤防

  您可以说“设备名称 + 布防/撤防”。

  比如：”智能门磁布防” 或 “智能门磁撤防“。

智能暗锁

布防/撤防

  您可以说“设备名称 + 布防/撤防”。

  比如：”智能暗锁布防” 或 “智能暗锁撤防“。

人体感应

布防/撤防

  您可以说“设备名称 + 布防/撤防”。

  比如：”人体感应布防” 或 “人体感应撤防“。

网关

布防/撤防

  您可以说“设备名称 + 布防/撤防”。

  比如：”网关布防” 或 “网关撤防“。

甲醛

数据刷新

  您可以说“设备名称 + 刷新/采集/更新”。

  比如：”甲醛刷新” 或 “甲醛采集“。

控制情景模式

  您可以说“执行/切换到/切到 + 情景模式名称”。

  比如：”执行回家模式“。

控制标签

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关/布防/撤防 + 标签 + 标签里面的设备名称”。

  比如：”打开办公室电脑插座“。(“办公室”是标签名称，“电脑插座”是标签“办公室”里面的设备名称)；”布防办公室人体感应“。(“办公室”是标签名称，“人体感应”是标签“办公室”里面的设备名称)

  如果您要控制标签里面某一类设备，您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关/布防/撤防 + 标签名称 + 设备类型名称”。

  比如：”打开办公室的灯“。(“办公室”是标签名称，“灯”指标签“办公室”里面所有的灯)

控制灯组

打开/关闭

  您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 灯组名称”。

  比如：”打开灯组” 或 “关闭灯组“。

控制亮度

  您可以说“调亮/调暗/亮/暗/亮起来/暗下去/暗下来/升高/调高/调大/大/加/降低/调低/调小/小/减 + 灯组名称”。

  比如：”调亮灯组” 或 “调暗灯组“。

操作步骤

  点击声控进入语音控制界面。

  按住声控按钮开始说话。

  右侧会显示您输入的语音信号。比如“插座开启”，控制成功后会看到相应的提示信息。比如“收到，主人，已控制成功。”，已控制设备会显示在”已控制设备列表“中。

  您也可以点击?按钮查看帮助，根据提示的格式输入语音信息。
