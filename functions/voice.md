# 语音控制

&emsp;&emsp;您可以通过语音来控制智能设备、执行情景模式以及标签。


&emsp;&emsp;您可以按照下面列出的语音格式操作，提高语音识别的效率。

# 您可以这样说：

## 控制单个设备

### 空调

#### 打开/关闭

&emsp;&emsp;您可以说“打开/开启/开机/开/关闭/停止/关掉/关机 + 设备名称”。

&emsp;&emsp;比如：“打开空调” 或 “关闭空调”。

### 控制温度

&emsp;&emsp;您可以直接设置空调温度，也可以调高/调低温度，您可以说“调高/调大/大/加/调低/调小/小/减 + 设备名称”。

&emsp;&emsp;比如：“空调30摄氏度” 或 “调高空调”。

### 控制模式

&emsp;&emsp;您可以说“设备名称 + 制热/自动/送风/除湿/制冷/扫风”。

&emsp;&emsp;比如：“空调制热” 或 “空调除湿”。


### 插座

### 窗帘

&emsp;&emsp;比如：“打开窗帘” 或 “关闭窗帘”。

#### 调整窗帘

&emsp;&emsp;您可以说“升高、调高、调大、大、加、降低、调低、调小、小、减 + 设备名称”。

&emsp;&emsp;比如：”调高窗帘” 或 “降低窗帘“。

### 电视、机顶盒

#### 打开/关闭

&emsp;&emsp;比如：“打开电视” 或 “关闭电视”。

#### 换台

&emsp;&emsp;您可以说“设备名称 + 上一个台/下一个台/上个台/下个台/上一个频道/下一个频道/上个频道/下个频道/频道加/频道减/换台/换频道/切换频道”。

&emsp;&emsp;比如：“机顶盒下一个台” 或 “机顶盒频道加”。

#### 音量调节

&emsp;&emsp;您可以说“设备名称 + 取消静音/关闭静音/恢复声音/静音/升高/调高/调大/大/加/降低/调低/调小/小/减”。

&emsp;&emsp;比如：“机顶盒静音” 或 “机顶盒取消静音” 或 “机顶盒调高”。

### 控制风扇灯

#### 打开/关闭

&emsp;&emsp;如果要打开/关闭风扇灯，您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 设备名称”。

&emsp;&emsp;比如：“开启风扇灯” 或 “关闭风扇灯”。

&emsp;&emsp;如果要控制风速，您可以说“低风速/中风速/高风速 + 设备名称”。

&emsp;&emsp;比如：”风扇灯低风速” 或 “风扇灯高风速“。

### 开关面板

#### 打开/关闭

&emsp;&emsp;比如：”打开开关面板” 或 “关闭开关面板“。 

> 说明：这样操作后开关面板会全部打开或全部关闭。

&emsp;&emsp;如果您想控制某一路，您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 设备名称 + 某一路的名称”或"打开/开启/开机/开/关闭/停止/关掉/关机/关 + 某一路的名称"

&emsp;&emsp;比如：”打开开关面板第1路” 或 “打开第一路“。

## 控制情景模式

&emsp;&emsp;您可以说“执行/切换到/切到 + 情景模式名称”。

&emsp;&emsp;比如：”执行回家模式“。

## 控制标签

&emsp;&emsp;您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关/布防/撤防 + 标签 + 标签里面的设备名称”。

&emsp;&emsp;比如：”打开办公室电脑插座“。(“办公室”是标签名称，“电脑插座”是标签“办公室”里面的设备名称)；”布防办公室人体感应“。(“办公室”是标签名称，“人体感应”是标签“办公室”里面的设备名称)

&emsp;&emsp;如果您要控制标签里面某一类设备，您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关/布防/撤防 + 标签名称 + 设备类型名称”。

&emsp;&emsp;比如：”打开办公室的灯“。(“办公室”是标签名称，“灯”指标签“办公室”里面所有的灯)

&emsp;&emsp;比如：”调亮灯组” 或 “调暗灯组“。
