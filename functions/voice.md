# 语音控制

&emsp;&emsp;您可以通过语音来控制智能设备、执行情景模式。

&emsp;&emsp;点击<font style='color:#ff0000;font-size:20px'>声控</font>进入语音控制界面。

<img src="./images/voice/进入语音.png" width = "290" height = "515">

&emsp;&emsp;按住声控按钮开始说话。

<img src="./images/voice/按住说话.png" width = "290" height = "515">

&emsp;&emsp;右侧会显示您输入的语音信号。比如“插座开启”，控制成功后会看到相应的提示信息。比如“收到，主人，已控制成功。”

<img src="./images/voice/控制成功.png" width = "290" height = "515">

&emsp;&emsp;您也可以点击<font style='color:#ff0000;font-size:20px'>?</font>按钮查看帮助，根据提示的格式输入语音信息。

<img src="./images/voice/语音帮助.png" width = "290" height = "515">

<img src="./images/voice/帮助界面.png" width = "290" height = "515">

&emsp;&emsp;你可以这样说：

&emsp;&emsp;如果是设备，您可以按照"设备名称+执行动作"或"执行动作+设备名称"方式说话，比如：插座开启或者开启插座。

&emsp;&emsp;如果是情景模式或者联动，您可以说"执行+情景模式名称/联动名称"，比如：执行回家。

| 类型           | 关键字      |
| ------------- |:-------------:|
| 情景模式、联动 | 执行、切换到、切到      |

&emsp;&emsp;如果是标签，您可以说"执行动作+标签名称+设备名称"，比如：关闭离家了风扇(“离家了”是标签名称)。

| 类型           | 关键字      |
| ------------- |:-------------:|
| 标签 | 打开、开启、开机、开、关闭、停止、关掉、关机、关、布防、撤防      |
