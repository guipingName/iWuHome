# 语音控制

&emsp;&emsp;您可以通过语音来控制智能设备、执行情景模式以及标签。可以参考下面的格式以提高语音识别的效率。

# 您可以这样说：

## 1. 控制单个设备

### 空调插座

#### 打开/关闭

&emsp;&emsp;您可以说“打开/开启/开机/开/关闭/停止/关掉/关机 + 空调插座名称”。

&emsp;&emsp;比如空调插座名称为“客厅空调”，您可以说“打开客厅空调” 或 “关闭客厅空调”。

### 控制温度

&emsp;&emsp;您可以设置到指定温度，比如20度，也可以调高/调低温度，您可以说“调高/调大/大/加/调低/调小/小/减 + 空调插座名称”。

&emsp;&emsp;比如空调插座名称为“客厅空调”，您可以说“客厅空调20度” 或 “调高客厅空调”。调高/调低时每次的变化量为1度，达到极值后设置无效。

### 控制模式

&emsp;&emsp;您可以说“空调插座名称 + 制热/自动/送风/除湿/制冷/扫风”。

&emsp;&emsp;比如空调插座名称为“客厅空调”，您可以说“客厅空调制热” 或 “客厅空调制冷”。

&emsp;&emsp;你也可以同时控制模式和温度，比如空调插座名称为“客厅空调”，您可以说“客厅空调制热20度”。

### 智能插座

#### 打开/关闭

&emsp;&emsp;您可以说“打开/开启/开机/开/关闭/停止/关掉/关机 + 智能插座名称”。

&emsp;&emsp;比如智能插座名称为“冰箱插座”， 您可以说“开启冰箱插座” 或 “关闭冰箱插座”。

### 灯

#### 打开/关闭

&emsp;&emsp;您可以说“打开/开启/开机/开/关闭/停止/关掉/熄/灭/关机/关 + 灯名称”。

&emsp;&emsp;比如灯的名称为“书房灯”，您可以说“打开书房灯” 或 “关闭书房灯”。

#### 控制亮度

&emsp;&emsp;您可以设置到指定亮度，比如80%，也可以调高/调低亮度，您可以说“调亮/调暗/亮/暗/亮起来/暗下去/暗下来/升高/调高/调大/大/加/降低/调低/调小/小/减 + 灯名称”。

&emsp;&emsp;比如灯的名称为“书房灯”，您可以说“调亮书房灯” 或 “书房灯亮起来”，调亮/调暗时每次的变化量为10%，达到极值后设置无效。您还可以说“书房灯80”，灯的亮度直接设置成80%。

### 开关面板

#### 打开/关闭

&emsp;&emsp;您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 开关面板名称”。

&emsp;&emsp;比如开关面板名称为“客厅开关面板”，您可以说“打开客厅开关面板” 或 “关闭客厅开关面板”。 执行打开/关闭时开关面板将每一路都打开或每一路都关闭。

&emsp;&emsp;如果只想控制开关面板中的某一路，您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 开关面板名称 + 某一路的名称”或"打开/开启/开机/开/关闭/停止/关掉/关机/关 + 某一路的名称"。

&emsp;&emsp;比如开关面板名称为“客厅开关面板”，您可以说“打开客厅开关面板第1路”，如果您的开关面板的每一路都命名了，比如第一路命名为“左侧电源”，第二路命名为“右侧电源”，您可以说“打开左侧电源” 或 “打开右侧电源”。

&emsp;&emsp;如果只想控制智能开关中的某一路，您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关 + 智能开关名称 + 某一路的名称”或"打开/开启/开机/开/关闭/停止/关掉/关机/关 + 某一路的名称"。

&emsp;&emsp;比如智能开关名称为“客厅智能开关”，您可以说“打开客厅智能开关第1路”，如果您的智能开关的每一路都命名了，比如第一路命名为“左侧电源”，第二路命名为“右侧电源”，您可以说“打开左侧电源” 或 “打开右侧电源”。

&emsp;&emsp;比如智能窗帘名称为“卧室窗帘”，您可以说“调高卧室窗帘” 或 “卧室窗帘70%”。

&emsp;&emsp;比如网关名称为“大厅”，您可以说“大厅布防” 或 “大厅撤防”。

## 2. 控制一类设备

&emsp;&emsp;如果有相同类型的设备，您可以同时控制。控制相同类型的设备和控制单个设备相似，只需要把设备名称换成设备类型名称。比如，空调插座类，类型名可以是“空调”、“空调插座”，您可以说“打开空调” 或 “打开空调插座”。执行后，当前家庭里的所有空调都会执行。但是如果您将某一个设备的名称改为默认类型名称(比如空调插座，命名为“空调”或“空调插座”)，那么当你说“打开空调”时就只有这一台设备打开，其它空调插座不会打开。下面列出了不同设备的设备类型名：

&emsp;&emsp;空调插座类：您可以说“空调”、“空调插座”、“迷你悟空”、“Mini悟空”、“悟空”

&emsp;&emsp;智能插座类：您可以说“插座”、“悟能”、“智能插座”

&emsp;&emsp;灯类：您可以说“调光灯”、“led灯”、“灯”、“魔灯”

&emsp;&emsp;开关面板类：您可以说“开关面板”

&emsp;&emsp;智能开关类：您可以说“智能开关”

&emsp;&emsp;网关类：您可以说“网关”

&emsp;&emsp;智能门磁类：您可以说“门磁”、“智能门磁”

&emsp;&emsp;智能暗锁类：您可以睡“门栓”、“门闩”、“智能暗锁”

&emsp;&emsp;人体感应类：您可以说“人体感应”、“红外感应”、“红外”、“红外人体”

&emsp;&emsp;窗帘类：您可以说“窗帘”、“卧室窗帘”

&emsp;&emsp;风扇灯类：您可以说“风扇灯”

&emsp;&emsp;遥控器类：您可以说“遥控器”

&emsp;&emsp;甲醛感应器类：您可以说“甲醛感应器”、“甲醛”、“甲醛传感器”

&emsp;&emsp;灯控盒类：您可以说“灯控盒”、“盒子”、“控制盒”

&emsp;&emsp;机顶盒类：您可以说“电视”、“机顶盒”

&emsp;&emsp;暖气阀类：你可以说“暖气”、“暖气阀”

&emsp;&emsp;声光报警器类：您可以说“声光报警器”


## 3. 控制情景模式

&emsp;&emsp;您可以说“执行/切换到/切到 + 情景模式名称”。

&emsp;&emsp;比如情景模式名称为“回家”，您可以说“执行回家” 或 “切换到回家”。

## 4. 控制标签

&emsp;&emsp;您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关/布防/撤防 + 标签 + 标签里面的设备名称”。

&emsp;&emsp;比如创建了一个名为“办公室”的标签，里面有一个名为“电脑插座”的智能插座和一个名为“体感”的人体感应器，如果想让插座通电，您可以说“打开办公室电脑插座”；如果想布防体感，您可以说“布防办公室体感”。

&emsp;&emsp;如果您要控制标签里面某一类设备，您可以说“打开/开启/开机/开/关闭/停止/关掉/关机/关/布防/撤防 + 标签名称 + 设备类型名称”。比如创建了一个名为“办公室”的标签，里面有多个智能插座，您可以说“打开办公室插座”。那么“办公室”里面所有的插座都会打开。

&emsp;&emsp;比如灯组名称为“客厅灯”，您可以说“调亮客厅灯” 或 “客厅灯亮度80%”。
