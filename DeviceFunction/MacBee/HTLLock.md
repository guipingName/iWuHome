# 汇泰龙门锁

&emsp;&emsp;在设备列表或标签界面点击云智能门锁显示输入密码框，输入正确密码后才能进入设备控制界面，将看到智能控制、用户管理、历史记录、提醒设置界面。

1. 智能控制：可以设置临时开锁密码。

	<img src="../images/MacBee/汇泰龙锁/开锁密码.png" width = "375" height = "667">
	
2. 用户管理：显示用户信息。
3. 历史记录：显示所有操作及报警记录。
4. 提醒设置：设置各种类型报警是否打开。
<img src="../images/MacBee/汇泰龙锁/报警.png" width = "375" height = "667">
